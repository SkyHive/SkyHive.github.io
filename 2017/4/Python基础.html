Python еҹәзЎҖ

еҸ‘иЎЁдәҺ 2017-04-07 еҲҶзұ»дәҺ Code йҳ…иҜ»ж¬Ўж•°пјҡ Walineпјҡ жң¬ж–Үеӯ—ж•°пјҡ 1.6k йҳ…иҜ»ж—¶й•ҝ ≈ 6 еҲҶй’ҹ

жңҖиҝ‘еңЁиҮӘеӯҰ PythonпјҢжүҖд»ҘдёҖиҫ№еӯҰд№ҹдёҖиҫ№жҖ»з»“дёҖдәӣзҹҘиҜҶзӮ№

В¶Python зҡ„иҫ“еҮә

Python зҡ„иҫ“еҮәе’Ң C иҜӯиЁҖзұ»дјјпјҢеңЁ print() еҮҪж•°йҮҢеҠ дёҠеӯ—з¬ҰдёІпјҲз”ЁеҚ•еј•еҸ·жҲ–иҖ…еҸҢеј•еҸ·пјҢдҪҶжҳҜдёҚиғҪж··з”ЁпјүпјҢдҫӢеҰӮпјҡ

>>> print('Hello world!')

print() еҮҪж•°д№ҹеҸҜд»ҘжҺҘеҸ—еӨҡдёӘеӯ—з¬ҰдёІпјҢз”ЁйҖ—еҸ·йҡ”ејҖеҚіеҸҜгҖӮprint() еҮҪж•°дҫқж¬Ўжү“еҚ°еӯ—з¬ҰдёІпјҢжҜҸйҒҮеҲ°йҖ—еҸ·е°ұдјҡиҫ“еҮәдёҖдёӘз©әж јгҖӮ
print() д№ҹеҸҜд»Ҙжү“еҚ°ж•ҙж•°жҲ–иҖ…и®Ўз®—з»“жһңпјҢдҫӢеҰӮпјҡ

>>> print('5+3=',5+3)

жіЁпјҡеҜ№дәҺ 5+3пјҢpython и§ЈйҮҠеҷЁдјҡиҮӘеҠЁи®Ўз®—еҮәз»“жһңпјҢдҪҶжҳҜ'5+3=' жҳҜеӯ—з¬ҰдёІиҖҢйқһж•°еӯҰе…¬ејҸ

В¶Python зҡ„иҫ“е…Ҙ

Pyhton жҸҗдҫӣдәҶ input() еҮҪж•°пјҢеҸҜд»Ҙи®©з”ЁжҲ·иҫ“е…Ҙеӯ—з¬ҰдёІпјҢе№¶дё”еӯҳеҲ°дёҖдёӘеҸҳйҮҸйҮҢпјҢдҫӢеҰӮпјҡ

>>> info = input()

иҫ“е…Ҙд»»ж„Ҹеӯ—з¬ҰеҗҺжҢүдёӢеӣһиҪҰе®ҢжҲҗиҫ“е…ҘпјҢиҫ“е…Ҙзҡ„еҶ…е®№е°ұиў«еӯҳж”ҫеңЁеҸҳйҮҸ info йҮҢйқўдәҶгҖӮ
input() еҮҪж•°еҸҜд»ҘжҳҫзӨәдёҖдёӘеӯ—з¬ҰдёІжқҘжҸҗзӨәз”ЁжҲ·пјҢеҰӮпјҡinput('please input something:')

В¶Python зҡ„ж•°жҚ®зұ»еһӢ

  • ж•ҙж•°пјҡPython еҸҜд»ҘеӨ„зҗҶд»»ж„ҸеӨ§е°Ҹзҡ„ж•ҙж•°пјҢеҢ…жӢ¬иҙҹж•ҙж•°гҖӮ

  • жө®зӮ№ж•°пјҡдёҺ C иҜӯиЁҖдёҖж ·пјҢеҸҜд»Ҙз”Ёж•°еӯҰжі•д№ҹеҸҜд»Ҙз”Ёз§‘еӯҰи®°ж•°жі•пјҲжҠҠ 10 з”Ё e жӣҝд»ЈпјүиЎЁзӨәгҖӮ

  • еӯ—з¬ҰдёІпјҡеӯ—з¬ҰдёІдҪҝз”ЁеҚ•еј•еҸ·жҲ–иҖ…еҸҢеј•еҸ·жӢ¬иө·жқҘзҡ„д»»ж„Ҹж–Үжң¬пјӣеҰӮжһңж–Үжң¬дёӯеҢ…еҗ«' е’Ң " еҸҜд»Ҙз”ЁиҪ¬д№үеӯ—з¬Ұ \ жқҘиЎЁзӨәгҖӮ

  • еёғе°”еҖјпјҡдёҖдёӘеёғе°”еҖјеҸӘжңү True е’Ң False дёӨз§ҚеҖјпјҲжіЁж„ҸеӨ§е°ҸеҶҷпјүпјӣеёғе°”еҖјеҸҜд»Ҙз”Ё andпјҲдёҺиҝҗз®—пјүгҖҒorпјҲжҲ–иҝҗз®—пјүе’Ң notпјҲйқһиҝҗз®—пјүиҝҗз®—гҖӮ

  • з©әеҖјпјҡз©әеҖјжҳҜ Python йҮҢдёҖдёӘзү№ж®Ҡзҡ„еҖјпјҢз”Ё None иЎЁзӨәгҖӮNone дёҚиғҪзҗҶи§Јдёә 0пјҢеӣ дёә 0 жҳҜжңүж„Ҹд№үзҡ„ж•ҙж•°пјҢиҖҢ None жҳҜдёҖдёӘзү№ж®Ҡзҡ„з©әеҖјпјӣзұ»дјјдәҺ C иҜӯиЁҖйҮҢйқўзҡ„ NULLгҖӮ

  • еҸҳйҮҸпјҡе’Ң C иҜӯиЁҖзұ»дјјпјҢеҸҳйҮҸеҗҚеҝ…йЎ»жҳҜеӨ§е°ҸеҶҷиӢұж–ҮгҖҒж•°еӯ—е’ҢдёӢеҲ’зәҝзҡ„з»„еҗҲпјҢдё”дёҚиғҪз”Ёж•°еӯ—ејҖеӨҙпјӣеҰӮпјҡ

a = 123

иҝҷз§ҚеҸҳйҮҸжң¬иә«зұ»еһӢдёҚеӣәе®ҡзҡ„иҜӯиЁҖз§°дёәеҠЁжҖҒиҜӯиЁҖпјҢдёҺд№ӢеҜ№еә”зҡ„жҳҜйқҷжҖҒиҜӯиЁҖгҖӮйқҷжҖҒиҜӯиЁҖеңЁе®ҡд№үеҸҳйҮҸжҳҜеҝ…йЎ»жҢҮе®ҡеҸҳйҮҸзұ»еһӢпјҢеҰӮпјҡ

int a = 123

е’ҢйқҷжҖҒиҜӯиЁҖзӣёжҜ”пјҢеҠЁжҖҒиҜӯиЁҖжӣҙзҒөжҙ»гҖӮ

  • еёёйҮҸпјҡжүҖи°“еёёйҮҸе°ұжҳҜдёҚиғҪеҸҳзҡ„еҸҳйҮҸпјҢеңЁ Python дёӯпјҢйҖҡеёёз”Ёе…ЁйғЁеӨ§еҶҷзҡ„еҸҳйҮҸеҗҚиЎЁзӨәеёёйҮҸгҖӮ

В¶Python зҡ„ж•ҙж•°йҷӨжі•е’ҢеҸ–дҪҷ

  • /пјҡиҝҷз§Қж•ҙж•°зҡ„йҷӨжі•еҫ—еҲ°зҡ„з»“жһңжҳҜжө®зӮ№ж•°пјҢеҚідҪҝдёӨдёӘж•ҙж•°жҒ°еҘҪж•ҙйҷӨз»“жһңд№ҹжҳҜжө®зӮ№ж•°гҖӮ

  • //пјҡиҝҷз§ҚйҷӨжі•д№ҹз§°дёәең°жқҝйҷӨпјҢдёӨдёӘж•ҙж•°зҡ„зӣёйҷӨз»“жһңд»Қз„¶дёәж•ҙж•°пјҢе’Ң C иҜӯиЁҖдёӯзҡ„ж•ҙж•°йҷӨжі•дёҖж ·пјҢж•ҙж•°йҷӨжі•з»“жһңдёәе°Ҹж•°ж—¶еҸ–ж•ҙж•°йғЁеҲҶгҖӮ

  • %пјҡдёҺ C иҜӯиЁҖзҡ„еҸ–дҪҷдёҖж ·пјҢз»“жһңдёәдёӨж•ҙж•°зӣёйҷӨзҡ„дҪҷж•°гҖӮ

В¶Python зҡ„ list

list зұ»дјјдәҺж•°з»„зҡ„жҰӮеҝөпјҢеҸҜд»ҘйҡҸж—¶ж·»еҠ е’ҢеҲ йҷӨе…ғзҙ пјӣдёҺж•°з»„дёҖж ·пјҢlist[] зҡ„зҙўеј•жҳҜд»Һ 0 ејҖе§Ӣзҡ„пјҢдҪҶжҳҜи®ҝй—®е…ғзҙ ж—¶иҰҒзЎ®дҝқзҙўеј•дёҚиҰҒи¶Ҡз•ҢпјҢеҸҜд»Ҙз”Ё len() еҮҪж•°жқҘиҺ·еҸ–е…ғзҙ зҡ„дёӘж•°пјҢеҰӮпјҡ

>>> len(list)       #еҸҳйҮҸlistжҳҜдёҖдёӘlist

жүҖд»ҘжңҖеҗҺдёҖдёӘе…ғзҙ зҡ„зҙўеј•жҳҜ len(list)-1пјҢеҪ“з„¶д№ҹеҸҜд»Ҙз”Ё -1 жқҘеҒҡзҙўеј•пјҢзӣҙжҺҘиҺ·еҸ–жңҖеҗҺдёҖдёӘе…ғзҙ  list[-1]пјҢд»ҘжӯӨзұ»жҺЁеҸҜд»Ҙз”Ё -2гҖҒ-3 жқҘиҺ·еҸ–еҖ’ж•°з¬¬дәҢдёӘзҡ„гҖҒеҖ’ж•°з¬¬дёүдёӘе…ғзҙ гҖӮ

Python жҸҗдҫӣ lsit.insert() еҮҪж•°е°Ҷе…ғзҙ жҸ’е…ҘеҲ°зү№е®ҡзҡ„дҪҚзҪ®пјҢеҰӮпјҡ

>>> list.insert(x,'info')#xдёәжғіжҸ’е…Ҙе…ғзҙ дҪҚзҪ®зҡ„зҙўеј•еҸ·пјҢinfoдёәжғіжҸ’е…Ҙзҡ„ е…ғзҙ 

д№ҹеҸҜд»Ҙз”Ё list.addend() еҮҪж•°е°Ҷе…ғзҙ жҸ’е…ҘеҲ°жң«е°ҫпјҢеҰӮпјҡ

>>> list.addend('info')     #infoдёәжғіжҸ’е…Ҙзҡ„е…ғзҙ 

жғіиҰҒеҲ йҷӨ list жң«е°ҫе…ғзҙ з”Ё list.pop() еҮҪж•°пјҢжғіиҰҒеҲ йҷӨжҢҮе®ҡдҪҚзҪ®зҡ„е…ғзҙ пјҢз”Ё list.pop(x) ж–№жі•пјҢе…¶дёӯ x жҳҜзҙўеј•дҪҚзҪ®гҖӮlist еҸҜд»ҘзӣҙжҺҘиөӢеҖјпјҢиҖҢдё” list дёӯзҡ„е…ғзҙ зұ»еһӢд№ҹеҸҜд»ҘдёҚеҗҢгҖӮ

В¶Python зҡ„ tuple

иҝҷжҳҜ Python зҡ„еҸҰдёҖдёӘжңүеәҸеҲ—иЎЁпјҢеҸ«еҒҡе…ғз»„пјҲtupleпјүгҖӮtuple е’Ң list зұ»дјјпјҢдҪҶжҳҜ tuple дёҖж—ҰеҲқе§ӢеҢ–е°ұдёҚиғҪиҝӣиЎҢдҝ®ж”№дәҶпјӣжҲ‘д»¬дҫқз„¶еҸҜд»ҘдҪҝз”Ё tuple[0]гҖҒtuple[-1] жқҘиҺ·еҸ–е…ғзҙ гҖӮ
з”ұдәҺ tuple дёҚеҸҜеҸҳпјҢжүҖд»ҘзӣёжҜ” list жӣҙеҠ е®үе…ЁпјҢжүҖд»ҘиғҪз”Ё tuple е°ұе°ҪйҮҸдҪҝз”Ё tupleгҖӮдёӢйқўиҜҙдёҖдёӢ tuple зҡ„е®ҡд№үпјҡ

>>> tuple = (1,2,3,4)       #жӯЈеёёзҡ„tupleе®ҡд№ү
>>> tuple = ()      #е®ҡд№үз©әзҡ„tuple

дҪҶжҳҜеҪ“е®ҡд№үдёҖдёӘе…ғзҙ зҡ„ tuple зҡ„ж—¶еҖҷпјҢеҰӮжһңеҶҷжҲҗиҝҷж ·пјҡ

>>> tuple = (1)

з”ұдәҺе°ҸжӢ¬еҸ· () ж—ўеҸҜд»ҘиЎЁзӨә tuple еҸҲеҸҜд»ҘиЎЁзӨәж•°еӯҰе…¬ејҸдёӯзҡ„е°ҸжӢ¬еҸ·пјҢе°ұдә§з”ҹдәҶжӯ§д№үгҖӮеӣ жӯӨпјҢPython и§„е®ҡиҝҷз§Қжғ…еҶөдёӢ () жҢүз…§ж•°еӯҰе…¬ејҸзҡ„е°ҸжӢ¬еҸ·и®Ўз®—пјҢз»“жһңеҚідёә 1пјҢжүҖд»ҘпјҢеҸӘжңүдёҖдёӘе…ғзҙ зҡ„ tuple е®ҡд№үжҳҜиҰҒеҠ дёҖдёӘйҖ—еҸ· , жқҘж¶ҲйҷӨжӯ§д№үпјҡ

>>> tuple = (1,)

еҪ“з„¶дәҶпјҢtuple зҡ„е…ғзҙ дёӯеҰӮжһңеҢ…еҗ«дёҖдёӘ list зҡ„иҜқпјҢlist зҡ„е…ғзҙ жҳҜеҸҜд»Ҙж”№еҸҳзҡ„

В¶Python зҡ„жқЎд»¶еҲӨж–ӯ

е’Ң C иҜӯиЁҖзұ»дјјпјҢPython зҡ„жқЎд»¶еҲӨж–ӯзҡ„иҜӯжі•еҰӮдёӢпјҡ

if <еҲӨж–ӯжқЎд»¶1>:
    <жү§иЎҢд»Јз Ғ>
elif <еҲӨж–ӯжқЎд»¶2>:
    <жү§иЎҢд»Јз Ғ>
elif <еҲӨж–ӯжқЎд»¶3>:
     <жү§иЎҢд»Јз Ғ>
        вҖҰвҖҰвҖҰвҖҰ
else:
     <жү§иЎҢд»Јз Ғ>

жҲ‘д»¬еҸҜд»ҘйҖҡиҝҮдҪҝз”Ёд№ӢеүҚиҜҙиҝҮзҡ„ input() еҮҪж•°зҡ„иҫ“е…ҘиҝӣиЎҢжқЎд»¶еҲӨж–ӯпјҢдҪҶжҳҜжңүдёҖдёӘйңҖиҰҒжіЁж„Ҹзҡ„ең°ж–№пјҢinput() еҮҪж•°иҝ”еӣһзҡ„ж•°жҚ®зұ»еһӢжҳҜ strпјҢдёҚиғҪзӣҙжҺҘе’Ңж•ҙж•°жҜ”иҫғпјҢжүҖд»Ҙеҝ…йЎ»иҰҒе…ҲдҪҝз”Ё int() еҮҪж•°е°Ҷ str иҪ¬жҚўдёәж•ҙж•°гҖӮ

В¶Python зҡ„еҫӘзҺҜ

В¶forвҖҰвҖҰin:

иҝҷдёӘеҫӘзҺҜеҸҜд»Ҙе°Ҷ list жҲ– tuple дёӯзҡ„е…ғзҙ йҒҚеҺҶеҮәжқҘпјҢзңӢдёӘдҫӢеӯҗпјҡ

num = ['1','2','3','4']
for x in num:
    print(x)

жү§иЎҢиҝҷж®өд»Јз ҒпјҢдјҡдҫқж¬Ўжү“еҚ°жҜҸдёҖдёӘе…ғзҙ пјҢжүҖд»Ҙ for x in вҖҰвҖҰ е°ұжҳҜжҠҠжҜҸдёӘе…ғзҙ еёҰе…ҘеҸҳйҮҸ xпјҢз„¶еҗҺжү§иЎҢзј©иҝӣеқ—зҡ„иҜӯеҸҘгҖӮжңүдәҶиҝҷдёӘеҫӘзҺҜжҲ‘д»¬е°ұеҸҜд»ҘеҒҡжұӮе’ҢдәҶпјҢдёҚиҝҮжҲ‘д»¬иҰҒд»Һ 1 еҶҷеҲ° 100 зЎ®е®һжңүзӮ№еӣ°йҡҫпјҢе№ёеҘҪ Python жҸҗдҫӣдәҶ range() еҮҪж•°пјҢеҸҜд»Ҙз”ҹжҲҗдёҖдёӘж•ҙж•°еәҸеҲ—пјҢеҶҚйҖҡиҝҮ list() еҮҪж•°еҸҜд»ҘиҪ¬жҚўдёә listгҖӮеҰӮпјҡ

>>> list(range(5))
[0,1,2,3,4]         #range(5)з”ҹжҲҗзҡ„жҳҜд»Һ0ејҖе§Ӣе°ҸдәҺ5зҡ„ж•ҙж•°
В¶while:

е’Ң C иҜӯиЁҖзұ»дјјпјҢжқЎд»¶ж»Ўи¶іе°ұеҫӘзҺҜпјҢжқЎд»¶дёҚж»Ўи¶іж—¶е°ұйҖҖеҮәеҫӘзҺҜгҖӮиҜӯжі•еҰӮдёӢпјҡ

while <еҲӨж–ӯжқЎд»¶>
<жү§иЎҢиҜӯеҸҘ>
В¶break е’Ң continue:

е’Ң C иҜӯиЁҖзұ»дјјпјҢbreak еҸҜд»ҘжҸҗеүҚйҖҖеҮәеҫӘзҺҜпјҢcontinue еҸҜд»Ҙи·іиҝҮеҪ“еүҚеҫӘзҺҜпјҢзӣҙжҺҘиҝӣе…ҘдёӢдёҖж¬ЎеҫӘзҺҜгҖӮ
